Five New Films

Програмне забезпечення дозволяє переглядати актуальну інформацію про кінострічки, які в поточний момент часу є в афіші кінотеатрів.


Встановлення програми:
Встановити програму можна двома способами:
1) Завантажити репозиторій на свій комп’ютер (за допомогою git pull або скачати zip-архів з GitHub) та зібрати продукт для свого телефону за допомогою Android SDK.
2) Завантажити apk файл з репозиторію ("FiveNewFilms.apk") на свій Android-пристрій та встановити просто відкривши файл (обов’язково потрібно дозволити встановлення додатків з непідтверджених джерел в налаштуваннях пристрою).

Запуск та робота програми:
Для запуску програми натисніть на іконку з зображенням кіноплівки і підписом "FiveNewFilms" в меню вашого пристрою, якщо з’єднання з інтернетом повільне, для запуску може знадобитися деякий час.
Як тільки програма почне працювати ви побачите коротку інформацію про 5 фільмів, які представлені у вигляді списку, список можна прокручувати за допомогою сковзання пальця по екрану пристрою. Біля кожного фільму можна натиснути кнопку "Деталі" та переглянути детальнішу інформацію. Із сторінки кожного фільму можна перейти на сайт "http://www.kinofilms.ua/" (на сторінку з відповідним фільмом) лише натиснувши на посилання знизу (відкриється встановлений на телефоні браузер, і завантажить вказану сторінку).

Вихід з програми:
Для виходу з програми використовуйте кнопку "назад" вашого пристрою.
